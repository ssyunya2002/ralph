# Research Skill

Web調査・情報収集タスクを実行する。

---

## The Job

1. 調査対象・目的を明確化
2. 複数の情報源から情報収集
3. 情報を構造化してまとめる
4. 出典を明記
5. 指定先に保存

---

## 調査手法

### 1. Web検索
WebSearchツールまたはTavily MCPを使用:

```
検索クエリ例:
- "{会社名} pricing 2025"
- "{業界} market trend 2025"
- "{製品カテゴリ} comparison review"
```

### 2. 情報源の優先順位

1. **公式情報**: 企業公式サイト、プレスリリース
2. **信頼性の高いメディア**: 業界専門メディア、ニュースサイト
3. **レビューサイト**: G2, Capterra, Trustpilot
4. **SNS・フォーラム**: X, Reddit, 専門フォーラム

### 3. 情報の検証

- 複数ソースで裏付けを取る
- 日付を確認（古い情報に注意）
- 一次情報と二次情報を区別

## タスクタイプ別のアプローチ

### 競合分析
```
調査項目:
- 価格帯・料金体系
- 主要機能・特徴
- ターゲット顧客
- 強み・弱み
- 最近の動向（資金調達、新機能等）
```

### 市場調査
```
調査項目:
- 市場規模・成長率
- 主要プレイヤー
- 顧客セグメント
- トレンド・変化
- 参入障壁
```

### トレンド調査
```
調査項目:
- 新技術・イノベーション
- 消費者行動の変化
- 規制・法改正
- 業界の課題
- 将来予測
```

## acceptanceCriteria対応

| Criteria | 確認方法 |
|----------|----------|
| 「情報源をN個以上引用」 | ## 情報源 セクションにN個以上のURLがある |
| 「表形式でまとめる」 | Markdown表が含まれている |
| 「結論を要約」 | ## 結論・示唆 セクションがある |
| 「指定ファイルに保存」 | ファイルが存在する |

---

## Tips

- **検索は英語も使う**: 海外情報は英語で検索すると情報量が多い
- **日付を確認**: 「2025」など年を検索に含める
- **公式ソースを優先**: 価格等は公式サイトを確認
- **スクリーンショット**: 重要な情報はスクショも検討
