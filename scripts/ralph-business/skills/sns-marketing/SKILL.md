# SNS Marketing Skill

SNSマーケティング戦略・実行プランを作成する。

---

## The Job

1. 目的・ターゲット確認
2. プラットフォーム選定
3. コンテンツ戦略策定
4. 投稿カレンダー作成
5. KPI設定

---

## プラットフォーム特性

### X (Twitter)
- **特徴**: リアルタイム性、拡散力
- **コンテンツ**: 短文、画像、スレッド
- **頻度**: 1-3回/日
- **ベストタイム**: 7-9時、12-13時、20-22時

### LinkedIn
- **特徴**: ビジネス特化、専門性
- **コンテンツ**: 長文、インサイト、事例
- **頻度**: 2-5回/週
- **ベストタイム**: 7-8時、12時、17-18時（平日）

### Instagram
- **特徴**: ビジュアル重視、ストーリー
- **コンテンツ**: 画像、リール、ストーリーズ
- **頻度**: 1回/日 + ストーリーズ
- **ベストタイム**: 11時、19-21時

### Facebook
- **特徴**: 幅広い層、コミュニティ
- **コンテンツ**: 記事、動画、イベント
- **頻度**: 1-2回/日
- **ベストタイム**: 13-16時

## コンテンツ戦略

### コンテンツピラー（柱）

```
例: B2B SaaS企業の場合
1. 教育コンテンツ (40%): ハウツー、Tips
2. ソートリーダーシップ (30%): 業界インサイト
3. 製品・サービス (20%): 機能紹介、事例
4. エンゲージメント (10%): 質問、投票
```

### コンテンツタイプ

| タイプ | 目的 | 例 |
|--------|------|-----|
| 教育 | 価値提供 | How-to、Tips、解説 |
| インスピレーション | 共感 | 名言、成功事例 |
| エンターテイメント | エンゲージメント | ミーム、クイズ |
| プロモーション | 販売 | 製品紹介、キャンペーン |
| UGC | 信頼構築 | お客様の声、事例 |

## Tips

- **一貫性が重要**: 不規則より定期的な投稿
- **80/20ルール**: 80%価値提供、20%宣伝
- **エンゲージメント優先**: いいね/RTより会話を重視
- **分析→改善**: データを見て継続的に調整
