# Sales Strategy Skill

セールス戦略・営業計画を立案する。

---

## The Job

1. 現状分析（調査結果の参照）
2. 目標設定
3. 戦略策定
4. アクションプラン作成
5. KPI定義

---

## 戦略立案フレームワーク

### 1. 現状分析
前タスクの調査・分析結果を参照:

- 市場環境
- 競合状況
- 自社の強み・弱み
- 顧客ニーズ

### 2. 目標設定（SMARTゴール）

- **Specific**: 具体的
- **Measurable**: 測定可能
- **Achievable**: 達成可能
- **Relevant**: 関連性がある
- **Time-bound**: 期限がある

例: 「Q2末までに新規顧客20社獲得、売上1,000万円達成」

### 3. 戦略の方向性

```
検討軸:
- ターゲット顧客（誰に）
- 提供価値（何を）
- チャネル（どうやって）
- 差別化ポイント（なぜ選ばれるか）
```

### 4. アクションプラン

```
各アクションに以下を定義:
- What: 何をするか
- Who: 誰が担当か
- When: いつまでに
- How: どうやって
- KPI: 成果指標
```

## acceptanceCriteria対応

| Criteria | 確認方法 |
|----------|----------|
| 「月別マイルストーン」 | 月ごとのセクションとアクションがある |
| 「KPIをN個以上」 | KPIテーブルにN行以上 |
| 「アクションアイテムN個以上」 | アクションプランにN個以上の項目 |
| 「前タスクを踏まえる」 | 現状分析が前タスクを参照している |

---

## Tips

- **データに基づく**: 「思う」ではなく「データによると」
- **アクションは具体的に**: 「営業強化」ではなく「週5件のコールドコール」
- **担当と期限を明確に**: 実行可能性を高める
- **振り返りを組み込む**: 週次/月次レビューの予定も入れる
